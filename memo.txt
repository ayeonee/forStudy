p3.
-HTML : Hyper Text Markup Langage 
-하이퍼 텍스트 : 하이퍼 링크 이용해 독자가 원하는 순서에 따라 기존문서에서 다른 문서로 접근할 수 있는 텍스트.
-마크업 : 태그를 이용하여 문서나 데이터의 구조를 명시하는 언어. 문서나 데이터의 구조를 표현.
-종합하면, HTML은 웹페이지의 구조 혹은 데이터 작성을 위한 마크업 언어. ex.<제목>HTML 배우기<제목>

p4.
-CSS : Cascading Style Sheets 
-스타일 시트 : 웹페이지의 스타일, 디자인을 정의하여 정리한 문서
-캐스캐이딩 : 폭포가 되어 떨어지다. CSS에 적혀있는 스타일을 우선순위에 따라 화면에 적용. 우선순위가 높은 스타일부터 낮은 스타일까지 순차적용.

p.15
<head> : HTML 문서에 대한 정보를 담고 있음, 일반 사용자는 웹 사이트에서 <head> 안에 속한 정보를 볼 수 없음.
	HTML 문서 안에 단 하나 존재, <html>태그 바로 아래 존재. <head>안에 담을 수 있는 태그는
	<meta> : 문서와 관련된 정보를 담는 태그. 브라우저만 읽을 수 있음. 
	<title> : 웹 페이지의 제목을 담는 태그. 브라우저에서 탭의 제목이나 즐겨찾기했을 때 이름으로 볼 수 있음.

<body> : HTML 문서 중 사용자에게 실제로 보여지는 부분. 문서내 단 하나 존재. 
	<h1> : 제목을 표시하는 태그. 1~6까지 중요도. 1이 가장 높음.
	<p> : 단락을 표시하는 태그. 글, 본문에 해당하는 내용 포함.
	이외에도 다양한태그 포함 가능.
<head> 태그 내부와 <body> 태그 내부에 존재하는 태그는 종류에 상관없이 여러번 사용 가능.

p.16 기본적인 레이아웃
시맨틱 태그(Semantic tag) : 의미를 가지는 태그. 나눠진 구획의 기능을 태그 이름만으로도 이해 가능.
	<header> : 웹페이지 혹은 <section>의 소개나 제목을 담기 위해 사용.
	<nav> : 내비게이션 역할을 수행하는 요소. 페이지 이동을 위한 메뉴를 주로 담음.
	<section> : 기준에 따라 구획을 구분하기 위해 사용. 기준에 맞는 <article>들을 담음.
	<article> : 주 내용을 담기 위해 사용.
	<aside> : 광고 또는 사이트의 주변 부분에 해당하는 내용을 담기 위해 사용.
	<footer> : 웹사이트 가장 아래에 들어가는 추가 정보를 담기 위해 사용.

p.19 제목과 본문 태그
제목 태그(Heading tag) : 제목을 나타내고 싶을 때 사용. <h1>~ <h6>, 일반적으로 대제목이 <h1>.
	제목 태그로 감싸는 것만으로도 다른 태그로 감싼 글씨와 다르게 보임. 이미 지정한 CSS가 있기 때문에.
본문태그 <p> : paragraphs의 약자, 단락 혹은 문단이란 뜻. 서로 이어지는 내용의 텍스트를 하나로 묶어 <p>로 감쌈.
본문태그 <br> : break의 약자. 엔터와 동일한 역할, 줄바꿈을 수행. <br> 태그가 적용되지 않는 <p> 태그는 내부에서 엔터를 입력해도 줄바꿈 적용되지 않음.
*빈 요소(Empty element) : <br>처럼 종료 태그를 쓰지 않아도 올바르게 인식하는 태그. 종료 태그가 없어서 컨텐츠가 비어 있고 태그 사이가 비어 있기 때문에 빈 요소라고 함.
본문태그 <pre> : preformatted의 약자. 형식화된 텍스트를 브라우저에 그대로 표시. 

p.20 나누기 위한 태그
<div> : division의 약자, '분할'. 아무것도 나타내지 않고 그저 각각의 태그를 구분짓기위해 사용되는 태그. 아무것도 나타내지 않고 그저 각각의 태그를 구분 짓기 위해 사용.
	굉장히 자주 쓰이며 보통 CSS를 이용하여 레이아웃을 배치할 때 CSS를 효과적으로 적용하기 위해, 또는 묶어서 사용하기 위해 많이 사용.

p.21 글자와 관련된 태그
<strong> : 태그로 감싼 단어 혹은 문장을 볼드체로 바꿈.
<em> : emphasized의 약자, 태그로 감싼 단어 혹은 문장을 이탤릭체로 바꿈.
<sup> : 태그로 감싼 단어 혹은 문장을 일반 위치보다 위로 올림. 
<sub> : 태그로 감싼 단어 혹은 문장을 일반 위치보다 아래로 내림.
<ins> : 태그로 감싼 단어 혹은 문장 아래 밑줄을 추가.(insert)
<del> : 태그로 감싼 단어 혹은 문장에 취소선을 추가.(delete)
